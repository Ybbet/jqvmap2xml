JQVMAP2XML

Convertisseur de cartes javascript JQVMAP

JQVMap est un plugin jQuery de rendus de cartes vectorielles. Il utilise du SVG, redimensionnable pour les navigateurs modernes comme Firefox, Safari, Chrome, Opera et Internet Explorer 9+.

Pour les besoins du CMS SPIP, un plugin a été créé pour pouvoir utiliser cette librairie et ainsi avoir un rendu de tous fichiers SVG. Mais pour cela, il est nécessaire de les convertir de SVG à un format propre à JQVMAP.
